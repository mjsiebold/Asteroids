
Asteroids.exe

Currently only Head-To-Head shared keyboard play is available.
Firing is slow to make it easier to evade in this one on one game.

Player 1
----------------------------
Color: Red
Keyboard Controls:
Turn Left.....z
Turn Right....x
Thrust........s
Fire..........Tab

Player 2
----------------------------
Color: Green
Keyboard Controls:
Turn Left.....left arrow
Turn Right....right arrow
Thrust........up arrow
Fire..........slash
